üìã Setup Instructions

Step 1: Create SharePoint List

  1. Go to your SharePoint site
  2. Click "New" ‚Üí "List"
  3. Name it "BOM_List"
  4. Add these columns:
    • ItemCode (Single line of text)
    • Description (Single line of text)
    • Category (Choice: Equipment, Materials, Services, Spare Parts)
    • PartNumber (Single line of text)
    • Unit (Choice: EA, SET, M, DAY, LOT)
    • Quantity (Number)
    • UnitPrice (Currency)

Step 2: Get SharePoint Site URL

  1. Go to your SharePoint site
  2. Copy the URL (e.g., https://yourcompany.sharepoint.com/sites/yoursite)
  3. Paste it in the "SharePoint Site URL" field above

Step 3: Connect

  1. Enter your SharePoint Site URL
  2. Enter "BOM_List" as the List Name
  3. Click "Connect"
  4. Sign in with your Microsoft account if prompted

  • All team members need at least "Contribute" permission to the SharePoint list
